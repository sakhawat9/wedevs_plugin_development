Display view count for specific posts using shortcode.

== Description ==

Smart Post View Count plugin allows you to display the view count for specific posts in WordPress. It provides shortcodes to easily integrate the view count into your posts and pages. Additionally, it adds a sortable column in the WordPress admin dashboard to sort posts based on their view count.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smart-related-post.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Plugin features:

- Display view count for individual posts using shortcodes.
- Admin dashboard integration for view count column.
- Sortable view count column in the WordPress admin post dashboard.
- Fully translatable.


Plugin Usage

Shortcodes :

- [spvc_post_view_count] Display the view count for the current post.
- [spvc_post_view_count id="POST_ID"] Display the view count for a specific post by ID.
