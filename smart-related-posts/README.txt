Display related posts in the same category.

== Description ==

Smart Related Post is a WordPress plugin that enhances your content by displaying related posts in the same category at the end of each post.

Plugin features:

- Easily customize the args with filters.
- Supports both grid and list layouts.
- Can change the number of columns to display in the grid layout.
- You can remove the thumbnail category and excerpt.
- Custom styles for a beautiful display.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `smart-related-post.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
